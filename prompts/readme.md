# Prompt Library

This directory contains reusable prompt documents grouped by purpose.

## Layout

- `docs/`
  - Documentation writing and cleanup prompts
- `session/`
  - Prompts used at the start or end of a coding session
- `git/`
  - Git and pull request related prompts
- `issues/`
  - Issue and task generation prompts
- `hardware/`
  - Hardware document generation and summary prompts

## Notes

- Keep prompt files focused on one workflow
- Prefer generic prompt names when the prompt can be reused across projects
- Keep hardware generation prompts separate from repo-specific coding prompts
